form teg의 속성

  • action : 입력한 정보를 제출할 곳(http://localhost/login.php--사용하는컴퓨터)
  • method="get" 입력한 값이 주소창에 보여주며 url로 전송
  • method="post" 입력한 값이 주소창에 보여주지 않고 숨겨서 전송(일반적방법)
  • label : 이름표. label for 와 input id의 id를 같이주어 짝을지어준다
  • input : 정보입력
  • label로 묶어주고 label의 text를 클릭해도 자동으로 커서가 input 으로 간다
  • autocomplete=on (자동완성기능이 on 예:입력해놓은 비번이나 id가 자동으로 보여진다)

    • input teg의 속성

  • placeholder(input teg 안에 표시되는 것)
  • autofocus (화면이 켜졌을데 자동으로 커서가 이곳에 위치)
  • required (반드시 입력)
  • pattern ([]-입력형식 명명, [.] 문자, [.+] 하나이상의 문자)
  • pattern="[a-zA-Z].+[0-9]" (첫번째 반드시 알파벳문자,중간-하나이상의 아무문자,마지막에 반드시 숫자)

    • textarea : 여러줄의 택스트를 입력

    button 속성

  • submit :클릭하면 입력한정보를 서버로전송 후 페이지 바뀜
  • button :입력값의 ui 만듬. 자바언어사용시 사용
  • reset : 입력한 값이 재설정된다
  • hidden : 프로그래머가 지정한 ***가 사용자의 눈에는 보이지않지만 전송버튼을 누르면 서버로 전송

  • 파일 업로드시 form teg에서 method="post" enctype="multipart/form-data"로 해줘야한다
  • input file 에서 name은 서버쪽 프로그래머가 원하는 이름을 지정해주어야한다
